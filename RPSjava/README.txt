# Rock Paper Scissors

## Overview
This is a simple Rock Paper Scissors game implemented in Java. The game allows users to play against the computer, making it a fun way to practice programming concepts and Java syntax.

## Features
- Play against the computer
- Randomized computer choices
- Simple command-line interface

## Getting Started
To run the game, ensure you have Java installed on your machine. You can download it from [Java's official website](https://www.java.com/en/download/).
