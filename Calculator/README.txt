# Calculator Project

This is a simple calculator application built as part of the Odin Project curriculum. The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- Addition, subtraction, multiplication, and division
- User-friendly interface
- Responsive design (if applicable)
- Error handling for invalid inputs

## Usage

- Enter numbers and select operations to perform calculations.
- Use the clear button to reset the calculator.

## Technologies Used

- HTML
- CSS
- JavaScript
